  • wsl命令详解

命令 含义
wsl或bash 打开默认子系统
wsl -d Ubuntu18 打开Ubuntu18子系统
wsl -l -o 可安装的有效分发的列表
wsl –install -d Ubuntu 安装默认Ubuntu
wsl –shutdown 关闭子系统
wsl –import Ubuntu-20.04 d:\ubuntu d:\ubuntu20.04.tar –version 2 导入已存的操作系统
wsl –export Ubuntu-20.04 d:\ubuntu20.04.tar 导出操作子系统
wsl -l -v 显示所有子系统
wsl -s Ubuntu-20.04 将分发版设置为默认值
bash -c “ls” windows命令提示符
wsl –unregister Ubuntu-20.04 注销分发版
wslconfig /s Ubuntu-18.04 修改默认子系统
